# 作业1答案

## 1. BERT文本分类和实体识别的关系及使用的loss

### 关系
在该代码中，BERT同时用于两个任务：
- **文本分类**：用于识别用户的意图（Intent Classification）
- **实体识别**：用于提取文本中的槽位信息（Slot Filling）

这是一个多任务学习模型，共享BERT的底层编码能力，同时完成两个相关但不同的任务。

### 各自使用的loss
- **文本分类（意图识别）**：
  - 使用`nn.CrossEntropyLoss()`
  - 输入：BERT的[CLS] token的池化输出（`pooler_output`）
  - 计算方式：直接对整个句子的意图类别进行分类

- **实体识别（槽位填充）**：
  - 使用`nn.CrossEntropyLoss()`
  - 输入：BERT的每个token的输出（`token_output`）
  - 计算方式：只计算有效token的loss（通过attention_mask过滤padding token）

## 2. 多任务训练 loss = seq_loss + token_loss 的坏处及处理方法

### 坏处
1. **训练不平衡**：如果两个任务的loss量级不同，可能会导致一个任务主导训练过程，影响另一个任务的性能
2. **任务重要性不同**：在实际应用中，可能某个任务（如意图识别）比另一个任务（如槽位填充）更重要
3. **收敛速度不同**：不同任务可能有不同的收敛速度，简单任务可能很快收敛，而复杂任务还需要更多训练

### 处理方法
1. **引入权重因子**：使用加权和的方式，如 `loss = α * seq_loss + β * token_loss`，其中α和β是可调整的超参数
2. **动态权重调整**：根据任务的性能动态调整权重，例如对性能较差的任务给予更高的权重
3. **标准化loss**：对每个任务的loss进行标准化处理，使它们在相同的量级上
4. **分阶段训练**：先单独训练每个任务，然后再进行联合训练
5. **梯度裁剪**：防止某个任务的梯度过大影响整体训练
